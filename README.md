
# Creative CV Portfolio - Africa Bulumko Jarana

A modern, interactive, and accessible portfolio website showcasing the professional journey of Africa Bulumko Jarana, a Software Developer and IT Support specialist with a unique perspective on accessibility and inclusive design.

![Portfolio Preview](Portfolio.jpg)

## 🌟 Overview

This portfolio is a comprehensive showcase of my skills, experience, and projects as a full-stack developer with expertise in React, C#, and enterprise IT support. Built with modern web technologies, it features smooth animations, dark/light mode support, and accessibility-first design principles.

## ✨ Features

### 🎨 Design & User Experience
- **Responsive Design**: Fully responsive across all devices and screen sizes
- **Dark/Light Mode**: Seamless theme switching with persistent preferences
- **Smooth Animations**: Powered by Framer Motion for engaging user interactions
- **Accessibility First**: Built with accessibility in mind, including sign language considerations
- **Modern UI**: Clean, professional design using shadcn/ui components

### 🚀 Technical Features
- **Interactive Hero Section**: 3D tilt effects and morphing backgrounds
- **Skills Visualization**: Animated progress circles and categorized skill display
- **Project Showcase**: Featured projects with video demonstrations and detailed descriptions
- **Experience Timeline**: Interactive timeline with animated elements
- **Contact Integration**: Direct links to professional profiles and communication methods

### 📱 Sections
- **Hero**: Introduction with profile image and contact links
- **About**: Personal story and professional values
- **Skills**: Technical and soft skills with proficiency levels
- **Projects**: Featured work including AI-powered applications
- **Technologies**: Comprehensive tech stack overview
- **Experience**: Professional journey and responsibilities
- **Education**: Academic background and certifications
- **Contact**: Contact information and professional references

## 📦 Installation & Setup

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn package manager

### Installation Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/EAJ1/creative-cv-portfolio.git
   cd creative-cv-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

5. **Preview production build**
   ```bash
   npm run preview
   ```
